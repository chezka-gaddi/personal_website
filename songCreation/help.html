CSS Modifications

Header Background Image

The header background image is a video with an image backup if the video is not able to render. The text is edited using the <h1> header tag to have the color of white and a different font from the rest of the page.

Navigation Bar

The navigation bar is created by editing an undordered list with hyperlink text to navigate from one page to another. When a link is active or is hovered over, the background color changes to indicate the interaction.

Content Layout

Two different layouts are used to create the website. For the index, a two column layout is created to divide the area where the music sheet resides and the area for the interactive buttons. This layout has the ability to restructure to have the contents to be stacked when the window is smaller than a certain value.

Buttons

The buttons used in the program has mutiple styling applied to them. Upon hovering on a button, the cursor changes to a pointer and a shadow affect is applied to indicate the button in focus. Styling is also applied to buttons that are disbaled to have a darker background and the cursor is changed to not-allowed.
